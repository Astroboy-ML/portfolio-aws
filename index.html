Portfolio AWSTerminé

Portfolio AWS S3 + CloudFront

Projet de conception et de déploiement d'un portfolio professionnel sur AWS reposant sur une architecture serverless sécurisée. L'infrastructure est distribuée via CloudFront, protégée par HTTPS avec ACM et déployée automatiquement grâce à un pipeline CI/CD GitHub Actions utilisant l'authentification OIDC. Le projet intègre également Terraform pour la gestion de l'infrastructure en Infrastructure as Code.
